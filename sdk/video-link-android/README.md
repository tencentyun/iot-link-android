### 概述
Video SDK 主要提供了设备P2P接入、实时监控对讲、本地回放等常带电IPC基本能力。


## Android 接入流程

1、在 App module下的build.gradle中添加依赖项

```
implementation 'com.tencent.iot.video:video-link-android:x.x.x'
```
具体版本号可参考 [video-link-android](TODO: sdk发布页url)
